# Детали расчёта налогов

## Общие принципы

### Базовые расчёты

```
Себестоимость (₽) = Выручка × (Процент себестоимости / 100)
Прочие расходы (₽) = Выручка × (Процент прочих расходов / 100)
Годовой ФОТ = Количество сотрудников × Оклад × 12
Страховые взносы = Годовой ФОТ × 30% (для всех режимов кроме АУСН)
```

### Расчёт НДС

```
Начисленный НДС = Выручка × Ставка НДС / (100 + Ставка НДС)

База закупок с НДС = Себестоимость × (Процент закупок с НДС / 100)
Входящий НДС (к вычету) = База закупок × Ставка НДС / (100 + Ставка НДС)

НДС к уплате = max(Начисленный НДС - Входящий НДС, 0)
```

## Режимы налогообложения

### 1. АУСН (Автоматизированная упрощённая система налогообложения)

**Ограничения:**
- Максимальная выручка: 60 000 000 ₽ в год
- Максимум сотрудников: 5 человек

**Ставки:**
- 8% (для некоторых видов деятельности)
- 20% (стандартная ставка)

**Особенности:**
- НДС не начисляется и не уплачивается
- Страховые взносы за сотрудников = 0 (уплачиваются автоматически)
- Налог = Выручка × Ставка

**Расчёт:**
```
Налог АУСН = Выручка × 8% (или 20%)
Страховые взносы = 0
НДС = 0
Чистая прибыль = Выручка - Расходы - Налог АУСН
```

### 2. УСН "Доходы" 6% + НДС 5%

**Применение с 2026 года**

**Расчёт:**
```
Налог УСН = Выручка × 6%
НДС к уплате = расчёт по общей формуле со ставкой 5%
Страховые взносы = ФОТ × 30%
Чистая прибыль = Выручка - Расходы - Налог УСН - НДС - Взносы
```

### 3. УСН "Доходы минус расходы" 15%

**Варианты:**
- С НДС 5%
- С НДС 22%

**Расчёт:**
```
Налоговая база = Выручка - Расходы
Налог УСН = max(Налоговая база, 0) × 15%
НДС к уплате = расчёт по общей формуле (5% или 22%)
Страховые взносы = ФОТ × 30%
Чистая прибыль = Выручка - Расходы - Налог УСН - НДС - Взносы
```

**Примечание:** Если расходы превышают доходы, налог = 0

### 4. ОСНО (Общая система налогообложения) + НДС 22%

**Расчёт:**
```
НДС к уплате = расчёт по общей формуле со ставкой 22%

База для налога на прибыль = Выручка - Расходы - НДС к уплате
Налог на прибыль = max(База, 0) × 20%

Страховые взносы = ФОТ × 30%
Чистая прибыль = Выручка - Расходы - НДС - Налог на прибыль
```

## Итоговые показатели

### Общая налоговая нагрузка (₽)

```
Налоговая нагрузка = Налог по режиму + НДС к уплате + Страховые взносы
```

### Налоговая нагрузка (%)

```
Налоговая нагрузка (%) = (Налоговая нагрузка (₽) / Выручка) × 100%
```

### Чистая прибыль

```
Чистая прибыль = Выручка - Расходы - Налоговая нагрузка
```

Где:
```
Расходы = Себестоимость + Аренда + Прочие расходы + ФОТ + Страховые взносы
```

## Упрощения модели

1. Не учитываются региональные особенности налогообложения
2. Ставки приняты как базовые (без повышающих коэффициентов при превышении лимитов)
3. Не учитывается налог на имущество, земельный налог и другие местные налоги
4. Упрощён расчёт НДС (в реальности может быть сложнее с различными ставками на разные товары/услуги)
5. Не учитывается возможность снижения УСН на страховые взносы
6. Страховые взносы ИП "за себя" не включены в расчёт

## Рекомендации по выбору режима

### АУСН выгоден если:
- Выручка до 60 млн ₽
- Сотрудников не больше 5
- Высокая маржинальность бизнеса
- Хотите минимум отчётности

### УСН Доходы 6% выгоден если:
- Расходы составляют менее 60% от выручки
- Много расходов, которые сложно документально подтвердить
- Хотите простоту учёта

### УСН Доходы-Расходы 15% выгоден если:
- Расходы составляют более 60% от выручки
- Все расходы можно документально подтвердить
- Готовы вести более сложный учёт

### ОСНО требуется если:
- Выручка превышает лимиты УСН
- Основные контрагенты работают с НДС и требуют выделения НДС
- Специфика деятельности не позволяет применять спецрежимы
