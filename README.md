# Калькулятор налогов для ИП

Одностраничное веб-приложение на Flask для расчёта и сравнения налоговой нагрузки при различных режимах налогообложения для индивидуальных предпринимателей в России.

## Функциональность

Калькулятор сравнивает следующие налоговые режимы:

- **АУСН 8%** (с ограничениями: выручка до 60 млн ₽, до 5 сотрудников)
- **АУСН 20%** (с теми же ограничениями)
- **УСН Доходы 6% + НДС 5%** (2026 год)
- **УСН Доходы минус расходы 15% + НДС 5%** (2026 год)
- **УСН Доходы минус расходы 15% + НДС 22%** (2026 год)
- **ОСНО + НДС 22% (ООО)**
- **ОСНО + НДС 22% (ИП)**

## Установка

### Вариант 1: Используя системный менеджер пакетов (Ubuntu/Debian)

```bash
sudo apt-get update
sudo apt-get install python3-flask
```

### Вариант 2: Используя pip

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python3 app.py
```

Затем откройте браузер и перейдите по адресу:

```
http://localhost:5005
```

## Использование

1. Заполните форму с данными о вашем бизнесе:
   - Выручка в год
   - Себестоимость (% от выручки)
   - Процент закупок с НДС
   - Аренда в год
   - Количество сотрудников и их оклад
   - Прочие расходы (% от выручки)

2. Нажмите кнопку "Рассчитать"

3. Изучите таблицу сравнения режимов налогообложения с показателями:
   - Налог по режиму
   - НДС к уплате
   - Страховые взносы
   - Общая налоговая нагрузка (в рублях и процентах)
   - Чистая прибыль

## Особенности

- Тёмная тема оформления
- Адаптивный дизайн для мобильных устройств
- Автоматический расчёт страховых взносов (30% от ФОТ для всех режимов кроме АУСН)
- Проверка лимитов для АУСН
- Форматирование чисел с разделителями тысяч

Это упрощённый калькулятор для ориентировочной оценки налоговой нагрузки. Для точных расчётов и принятия решений по выбору налогового режима рекомендуется проконсультироваться с бухгалтером или налоговым специалистом.
